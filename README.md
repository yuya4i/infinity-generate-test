# Infinity Generate - クーポン一覧アプリ

## 🤖 AI自律開発実験プロジェクト

このプロジェクトは、**AI（Claude）が自律的に機能を提案・設計・実装する実験**です。

### 実験の目的
人間の介入を最小限に抑え、AIが以下を自律的に行うことで、どこまで精度の高いシステムを構築できるかを検証します：
- ✅ 顧客ニーズの分析と機能提案
- ✅ アーキテクチャ設計
- ✅ コーディング実装
- ✅ テスト実行
- ✅ ドキュメント作成
- ✅ Git管理（ブランチ戦略、コミット）

### 開発プロセス
1. **マネージャーとしての思考**: ビジネス価値・顧客課題を分析
2. **機能提案**: 0→1で新機能を提案
3. **自動承認**: 人間の承認待ちなしで実装開始
4. **Featureブランチ開発**: 各機能ごとに独立したブランチで開発
5. **テスト駆動**: ブラウザテストで動作確認
6. **ドキュメント更新**: 実装内容を自動記録

---

Web上やアプリ上で取得可能なクーポンを集約し、一覧化して閲覧・利用できるアプリケーションです。

## 概要

本アプリは、ユーザーが見落としがちなクーポンを集約し、カテゴリ別に整理して表示することで、購買意欲の向上を目的としています。

### 主な機能

#### データ管理
- LocalStorageによるクーポンデータの永続化
- クーポン手動追加フォーム
- クーポンの個別削除
- 期限切れクーポンの一括削除
- JSONファイルによるエクスポート/インポート

#### 表示・フィルタリング
- クーポン一覧表示（カード形式）
- カテゴリ別フィルタリング（飲食、ショッピング、旅行、エンタメ、サービス、その他）
- 検索機能（タイトル、説明、取得元、タグで検索）
- ソート機能（取得日、有効期限で昇順・降順）

#### 統計・分析
- クーポン統計ダッシュボード
  - 総クーポン数
  - 有効なクーポン数
  - 7日以内に期限切れになるクーポン数
  - 期限切れクーポン数

#### UI/UX
- レスポンシブデザイン（モバイル対応）
- ダークモード対応
- 直感的なカードレイアウト
- 外部サイトへの遷移

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **パッケージマネージャー**: npm

## プロジェクト構成

```
├── app/                    # Next.js App Router
│   ├── layout.tsx          # ルートレイアウト
│   ├── page.tsx            # トップページ
│   └── globals.css         # グローバルスタイル
├── components/             # Reactコンポーネント
│   ├── CouponCard.tsx      # クーポンカード
│   ├── CategoryFilter.tsx  # カテゴリフィルタ
│   ├── AddCouponForm.tsx   # クーポン追加フォーム
│   ├── SearchBar.tsx       # 検索バー
│   ├── SortControl.tsx     # ソートコントロール
│   └── CouponStats.tsx     # 統計ダッシュボード

```

## 開発環境のセットアップ

### 必要要件

- Node.js 18以降
- npm

### インストール

```bash
# リポジトリのクローン
git clone git@github:yuya4i/infinity-generate-test.git
cd infinity-generate-test

# 依存関係のインストール
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

### ビルド

```bash
npm run build
```

### 本番環境での起動

```bash
npm start
```

## テスト

agent-browserを使用したE2Eテストを実行できます。

```bash
# agent-browserでテスト
agent-browser open http://localhost:3001
agent-browser snapshot -i
```
